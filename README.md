## HexClock

Extended version of [Mukesh Sankhla's RGB HexMatrix](https://hackaday.io/project/173732-rgb-hexmatrix-iot-clock).

### Features

- **Captive portal configuration**  
  Simple Wi-Fi and device setup through a built-in configuration portal.

- **Home Assistant integration**  
  Uses the [arduino-home-assistant](https://github.com/dawidchyrzynski/arduino-home-assistant) library for integration with Home Assistant.

- **Integrated time zone database**  
  Built-in time zone database allows accurate automatic time configuration without manual offset adjustments.
